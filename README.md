# upbit-websocket
프로그램 실행 환경
OS : MacOS Monterey 12.2.1
IDE : Visual Studio code
DB : MySql 8.0

프로그램 실행 프로세스
0. nodejs 사전 설치
1. > npm install 플러그인 설치
2. > tsc 타입스크립트 소스 빌드
3. > npm start

트러블슈팅

[ERROR] : Client does not support authentication protocol requested by server; consider upgrading MySQL client
